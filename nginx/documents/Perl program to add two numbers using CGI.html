Below is a simple Perl program that uses CGI to accept two integers as arguments from a web form, calculates their sum, and returns a well-formatted HTML page displaying the result:

```perl
#!/usr/bin/perl

use strict;
use warnings;
use CGI;

# Create a new CGI object
my $cgi = CGI->new;

# Get the values of the two integers from the form
my $num1 = $cgi->param('num1') || 0;
my $num2 = $cgi->param('num2') || 0;

# Calculate the sum
my $sum = $num1 + $num2;

# Print the HTML content with the result
print $cgi->header;
print $cgi->start_html(
    -title   => 'Sum of Two Integers',
    -bgcolor => 'white'
);
print $cgi->h1('Sum of Two Integers');
print $cgi->p("The sum of $num1 and $num2 is: $sum");
print $cgi->end_html;
```

Save this code in a file, say `sum.cgi`, and place it in a directory accessible by your web server. Ensure that the file has execute permissions (`chmod +x sum.cgi`).

Next, create an HTML form to send the integers to this CGI script:

```html
<!DOCTYPE html>
<html>
<head>
    <title>Sum Calculator</title>
</head>
<body>
    <h2>Enter two numbers to calculate their sum:</h2>
    <form action="/cgi-bin/sum.cgi" method="GET">
        <label for="num1">Number 1:</label>
        <input type="number" id="num1" name="num1"><br><br>
        <label for="num2">Number 2:</label>
        <input type="number" id="num2" name="num2"><br><br>
        <input type="submit" value="Calculate Sum">
    </form>
</body>
</html>
```

Save this HTML code in a file, say `index.html`, and place it in the root directory of your web server or wherever you wish to access it from your web browser.

When you access `index.html` in your web browser and enter two integers, it will send those integers to `sum.cgi`, which will calculate their sum and return a well-formatted HTML page displaying the result.
