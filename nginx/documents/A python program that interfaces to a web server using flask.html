Below is a simple Python program that uses Flask to create a web server. This server accepts two integers as arguments in the URL, calculates their sum, and returns a well-formatted HTML file displaying the result.

```python
from flask import Flask, request, render_template_string

app = Flask(__name__)

@app.route('/add')
def add():
    # Get the values of 'a' and 'b' from the query parameters
    a = request.args.get('a', type=int)
    b = request.args.get('b', type=int)

    # Calculate the sum
    if a is not None and b is not None:
        result = a + b
    else:
        result = None

    # Render HTML template with the result
    html_template = """
    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Sum Calculator</title>
    </head>
    <body>
        <h1>Sum Calculator</h1>
        <form action="/add" method="get">
            <label for="a">Enter first number:</label>
            <input type="number" id="a" name="a"><br><br>
            <label for="b">Enter second number:</label>
            <input type="number" id="b" name="b"><br><br>
            <input type="submit" value="Calculate">
        </form>
        {% if result is not none %}
        <h2>Result: {{ result }}</h2>
        {% endif %}
    </body>
    </html>
    """

    return render_template_string(html_template, result=result)

if __name__ == '__main__':
    app.run(debug=True)
```

Save this code in a file named `app.py`. Make sure you have Flask installed (`pip install flask`). Then, you can run this script, and it will start a web server on your local machine.

To access the sum calculator, go to `http://127.0.0.1:5000/add?a=<first_number>&b=<second_number>` in your web browser, replacing `<first_number>` and `<second_number>` with the integers you want to add. For example, `http://127.0.0.1:5000/add?a=3&b=5` will return the sum of 3 and 5.
