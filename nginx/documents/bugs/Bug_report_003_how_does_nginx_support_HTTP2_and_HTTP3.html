Bug report #3: One (or more) of the slides in module 2 should explain how to use HTTP/2 and HTTP/3

Priority: Medium

Author: Jeff Silverman

Date: 2024-02-20

There is no discussion about configuring NGINX for HTTP/2 or HTTP/3 in the slides.


Symptoms:


Analysis

Tecmint has How to Enable HTTP/2.0 in Nginx.  There is also an official page about the ngx_http_v2_module.  HTTP/2 is supported in version 1.9.5.  HTTP/3 is supported after 1.25.0, but see Support for QUIC and HTTP/3 in the official documentation.

There should also be some documentation on how to test.


Conclusion

Add a slide or two.
