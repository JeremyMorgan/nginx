Glossary and acronyms

.
.
CA
Certificate Authority
Cert or Certificate
A public key that an HTTPS server gives a client that the client uses to prove that the server has the private key.  The server can have the private key if and only if a CA gave the server the cert.
CSR
Certificate Signing Request
DNS
Domain Name Service - a distributed service on the internet that translates between names and addresses as well some other useful key value lookups.
GUID
Globally Unique Identifiers (See RFC 4122.).  These are 128 bit numbers that have a very, very high probability that they will be unique.  See UUID.  GUIDs are used by Microsoft, who just had to be different.  Most of the time, it doesn't matter.  Most of the time.
HTTP
HyperText Transport (RFC list...)
HTTPS
HyperText Transport with Security (RFC list...)
.
.
Registrar
An organization which has been approved to make entries in one or more DNS TLDs.
SNI
Server Name Indication  (RFC 6066) See the nginx.org official documentation.
SSL
Secure Sockets Layer.  SSL is an anachronism, use TLS instead.
TLD
DNS Top Level Domains.  The highest level names in the DNS name space.  Examples include .COM, .ORG, .UK, .NAME and .XXX.  IANA has a list of TLDs.
TLS
Transport Layer Security
UUID
Universally Unique Identifiers.  These are 128 bit numbers that have a very, very high probability that they will be unique.  See GUID.  See RFC 4122.  There are standard libraries in most programming langauges for using UUIDs.  Use them.

Zero-trust
A system design philosophy in which a server provider does not trust any inbound connection without some trusted thing that attests to its trustworthiness.
